# Bookbot
Bootbot is a python library for generating basic letter frequency report for books.

## Installation
```bash
git clone 
```

## Usage

1. Manually put the book.txt file under books/ 
2. Run the program.
```bash
python main.py
```
3. Check the bookbotreport.txt file, report should have been overwitten.
